# Computer Builder

A simple program demonstrating the **Builder design pattern** in constructing computers.

## Components
- `Computer` class
- `ComputerBuilder` interface
- Concrete builders (e.g., `GamingComputerBuilder`, `OfficeComputerBuilder`)
- `Director` class
